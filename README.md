# dotfiles


**Required**  

[nvim]
- install neovim
- install vim-jetpack  (curl -fLo ~/.config/nvim/autoload/jetpack.vim --create-dirs https://raw.githubusercontent.com/tani/vim-jetpack/main/plugin/jetpack.vim)
- :JetpackSync
- :TSInstall {all}
- :Mason
- install tree-sitter (use cargo/rustup)
- install nerd-font
